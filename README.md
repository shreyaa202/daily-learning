# 🤖 Webflow AI Content Manager
**Completed: 30-Day AI Engineering Challenge**

This is a full-stack AI application designed to help Webflow developers automate their CMS content generation. 

### ✨ Features
- **Secure Access:** Password-protected dashboard.
- **AI Personalities:** Choose between Luxury, Gen-Z, and Professional brand voices.
- **Data Insights:** Automatic inventory and pricing charts.
- **Bulk Export:** Process 100s of items and download a Webflow-ready CSV.
